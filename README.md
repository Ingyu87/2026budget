# 2026 AI·디지털 활용 선도학교 운영 인사이트

예산 집계와 에듀테크 선택 경향, 필수·선택과제 사례를 일반 교사가 쉽게 살펴볼 수 있도록 만든 반응형 웹앱입니다.

## 주요 기능

- 학교별 집행률 가운데 수준과 익명 분포
- 네 예산영역의 학교당 평균 계획액·집행액·집행률
- 학교명을 입력하지 않는 브라우저 내 자체진단
- 인터랙티브 에듀테크 Top 10 워드클라우드
- 기능 유형별 선택 경향과 2학기 운영 인사이트
- 2학기 운영 체크리스트
- 필수과제 1·2·3과 선택과제의 익명 정성 사례 분석
- 사례 흐름 → 인사이트 → 2학기 생각거리 상세 패널
- 예산 데이터 분석과 과제 사례 분석을 나누는 첫 진입 화면

## 개인정보 원칙

원본 CSV, 학교명, 학교별 금액, 원문 지출내역은 저장소와 배포 결과물에 포함하지 않습니다. 사례는 학교를 특정할 수 있는 표현을 덜어내고 공통 운영 방식 중심으로 재서술합니다.

## 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm test
```

## 배포

표준 Next.js App Router 프로젝트입니다. GitHub 저장소를 Vercel에 연결하면 `npm run build`가 `.next` 출력을 생성하고 `main` 브랜치 변경사항이 자동 배포됩니다.
