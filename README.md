# 2026 AI·디지털 선도학교 예산 레시피

58개 선도학교의 익명 예산 집계와 에듀테크 선택 경향을 일반 교사가 쉽게 살펴볼 수 있도록 만든 반응형 웹 대시보드입니다.

## 주요 기능

- 학교별 집행률 가운데 수준과 익명 분포
- 네 예산영역의 학교당 평균 계획액·집행액·집행률
- 학교명을 입력하지 않는 브라우저 내 자체진단
- 인터랙티브 에듀테크 Top 10 워드클라우드
- 기능 유형별 선택 경향과 2학기 운영 인사이트
- 2학기 운영 체크리스트

## 개인정보 원칙

원본 CSV, 학교명, 학교별 금액, 원문 지출내역은 저장소와 배포 결과물에 포함하지 않습니다. 사이트에는 58개교 익명 집계값만 포함됩니다.
